📈 Prospect Theory (Kahneman-Tversky)

Gains: u(r) = rα  ·  Losses: u(r) = −λ·|r|β. Captures diminishing sensitivity and loss aversion. Empirical defaults: α=β=0.88, λ=2.25.

Exponent for gains (0–1). Lower = more concave.
Exponent for losses (0–1). Lower = more convex.
How much more losses hurt than equivalent gains. 1 = neutral.

💹 Sortino Ratio

Like Sharpe but only penalises downside volatility — returns below the risk-free threshold. Sortino = (mean − rf) / downside deviation. Leaves upside variance unpunished.

E.g. 0.04 = 4%. Returns below this are penalised as downside.

Ω Omega Ratio

Ratio of probability-weighted returns above the threshold to probability-weighted returns below it. Omega > 1 means more upside than downside relative to the threshold. Uses the full return distribution, not just variance.

E.g. 0.04 = 4%. Returns above this count as gains, below as losses.

△ Convex Loss Utility

Identical to Prospect Theory on the gains side (concave, diminishing marginal utility). On the loss side, disutility increases with each additional unit of loss — large losses feel disproportionately painful. The default β = 1/α ≈ 1.136 mirrors the gain curve's shape while λ = 2.25 keeps the 2.25× loss-aversion scaling.

Concavity of the gains curve. 0.88 = Kahneman-Tversky default.
Convexity of the loss curve. Default 1/α ≈ 1.136 (geometric mirror of α).
How many times more the loss side weighs relative to gains. 2.25 = Kahneman-Tversky.
